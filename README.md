# Cинтез датасета


## Установка
```pip install requirements.txt```

Чтобы работало нужно чтобы в папке проекта был ffmpeg, 
чтобы не работало не нужно 


## Конфигурация
```config.yaml``` нужен для указания путей к данным и к ffmpeg

## Предобработка
```preprocess.py``` нужен для конвертации mp3/ogg/… → mono-16kHz WAV и удаления тишины

## Генерация
```generate_synthetic.py``` нужен для сборки синтетики по сценариям и вывода `.wav` + `.yaml`

## Сценарии
папка ```scenarios/``` содержит шаблоны сценариев в формате YAML
